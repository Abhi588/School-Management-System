PREREQUISITE;
1. MSAccess Database
2. Netbeans
3. Java -version 1.8.0 

If you want to access through cmd;
	java -version 1.8.0 should be installed { Mandatory };
	
	Step 1:- Open cmd
	Step 2:- Run the command
		java -jar "location of file\SchoolManagementSystem.jar"
	
		Example :- java -jar "C:\Users\Abhishek\Documents\NetBeansProjects\SchoolManagementSystem\dist\SchoolManagementSystem.jar"
	Step 3:- Copy database file location paste in text field for database location
			
			Example:-C:\Users\Abhishek\Documents\NetBeansProjects\SchoolManagementSystem
	
	Step 4:- Click on set Path {it will set your database file location}
	Step 5:- Type Login details as
		for Administrator :-
		User name:- Admin
		password:-  password
		for student:-
		Username :- Abhishek
		password:- kumar
	Step 6:-Use functions then close.
